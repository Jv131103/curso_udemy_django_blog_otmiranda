    Configurando o django-summernote como editor HTML em TextField nos models Django


Vamos instalar o django-summernote:

    Em requirements.txt vamos adicionar:

        Django>=5.1.7,<5.2
        psycopg2-binary>=2.9.10,<2.10
        Pillow>=9.5.0,<9.6
        django-summernote>=0.8.20.0,<0.8.21

Agora, vamos rebuildar o nosso docker sem gerar vários containers:

    docker compose up --build --remove-orphans --renew-anon-volumes --force-recreate

    from django.conf import settings
    from django.conf.urls.static import static
    from django.contrib import admin
    from django.urls import include, path

    urlpatterns = [
        path('', include('blog.urls')),
        path('summernote/', include('django_summernote.urls')),
        path('admin/', admin.site.urls),
    ]


    # Isso é importante, por que senão, não conseguimos ver os arquivos de media
    # enviados pelo nosso usuário em nosso desenvolvimento
    if settings.DEBUG:
        urlpatterns += static(
            settings.MEDIA_URL,
            document_root=settings.MEDIA_ROOT
        )
